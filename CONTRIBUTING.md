# Contributing to American Capitalism 2.0

Contributions are welcome through pull requests. All changes are transparent and
publicly attributed through Git history.

## Ground Rules

- All substantive text changes to the thesis must be proposed via Pull Request.
- Pull Requests must clearly describe:
  - What changed
  - Why it improves clarity, accuracy, structure, or coherence
- No anonymous edits: a GitHub account is required.
- Submitting a Pull Request does **not** grant any right to represent,
  reproduce, publish, commercialize, or politically deploy the content of this
  repository.

Expressing political support or alignment with American Capitalism 2.0 does not
grant permission to reproduce, modify, or integrate the thesis into political
operations or platforms. Only the author may authorize official political use.
